Guidelines to Operate this App

Features in this app allow You, to Add, Edit and View your parking(s) in ease without waiting in the queue. The app also allows you to make payments for the parking, displaying the rates and calculating them per hours making it easy for our Customers.

If you are first time user, kindly register to create an account by saving your personal info, payment info and Car info. All this can be done by selecting the “Sign-Up” option.

 To just get in to the app, Hit the Sign-In button and login with your credentials and Continue parking ! 

The Navigation bar to the left side of the App makes it easy for you to select whatever option you want to Opt.

  • Home
  • Receipt
  • Search
  • Settings
  • Contact Us


The Home menu will display two buttons for Adding a parking and View a parking. The add parking button takes you to the screen to add parking and logging in by registering a profile. The View parking button allows to view all the parkings done by You.

The Receipt menu will show all the previous transaction or Bills previously made on this App.

Search Menu will open the Maps and help you to locate nearest parking facility available.

Update Profile menu option will allow user to edit his/her personal info, car info and Payment info.

In the Contact Us menu, The Help option will allow the user to enter this website which shows the guidelines to navigate and use the app.  In the consumer support menu within Contact us, there are two options allowing the user to Contact the app developers through Phone and Email for queries.
